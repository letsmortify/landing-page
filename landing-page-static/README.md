# Sankalp - Taste of South | Dehradun Landing Page

A beautiful, responsive landing page for Sankalp Restaurant's Dehradun franchise.

## Features

- **Hero Section**: Award-winning headline with Book Now & Menu CTAs
- **Awards Marquee**: Guinness World Record Holder, Times Food Award Winner
- **Signature Dishes Gallery**: 4 authentic South Indian dishes
- **Customer Testimonials**: Real Google reviews
- **Location & Contact**: Address, phone, hours with Google Maps
- **Floating Offer**: 20% OFF on prior bookings

## Tech Stack

- React 19
- Tailwind CSS
- Lucide Icons
- Google Fonts (Playfair Display, Manrope, Italiana)

## Deployment

This is a static site ready for GitHub Pages deployment. Simply enable GitHub Pages in repository settings.
